
# Neuro-Aphasia Geometry Engine 🧠🛰️

Este projeto redefine o conceito de **Treinamento de IA**, migrando da generalização probabilística tradicional para a **Sintonia de Fase em Hiperplanos Geométricos**.

## 🚀 O Diferencial Tecnológico
Ao contrário dos modelos "caixa-preta", este sistema utiliza:
- **Especialistas de Campo:** Processamento distribuído (Spark) de Biofísica, Química e Topologia (Ricci).
- **Aceleração Intel MKL:** Álgebra vetorial de alta precisão para cálculo de fase (Seno/Cosseno).
- **Visualização Plexus:** Renderização em tempo real (openFrameworks) da trajetória de inferência.

## 🏗️ Arquitetura de Dados (Medalhão)
1. **Bronze:** Ingestão de Voxels e Farmacologia.
2. **Silver:** Particionamento Vertical (Geometria vs Química).
3. **Gold:** Catálogo de Microestrutura (Onde o 0,001% se torna visível).

## 🛠️ Como Executar
1. Configure o ambiente Conda: `conda activate neuro_afasia_env`
2. Gere a literatura: `python src/generate_partitions.py`
3. Sintonize a fase: `python src/root_cause_analysis.py`
4. Inicie o Oráculo: `python src/bridge_osc/sender.py`
5. Visualize no OF: Compile e execute o `braingraph` C++.

> *"O treinamento não é colocar mais pesos, é criar parâmetros para que o campo se ajuste ao valor real da função."*
